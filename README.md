# Scenario 
The manufacturing industry is focusing on automation using IoT devices and embedded controllers. The goal is to reduce manual monitoring, improve energy efficiency, and enable predictive maintenance.
Target audience: Manufacturing companies in automotive, electronics, and food processing sectors.
________________________________________

# Procedure and Prompt Examples

## 1. Comparative Analysis Prompt
Prompt Example:
“Compare and contrast the role of IoT-based automation with traditional manual monitoring in manufacturing industries, focusing on efficiency and predictive maintenance.”


Expected Response:
•	Side-by-side analysis showing IoT automation advantages (real-time monitoring, predictive analytics, energy efficiency) versus manual monitoring drawbacks (delays, higher downtime, limited optimization).
________________________________________

## 2. Experiential Perspective Prompt
Prompt Example:
“As a factory manager, explain how IoT-based predictive maintenance can reduce downtime and improve productivity.”
Expected Response:
•	Practical explanation from the manager’s viewpoint focusing on reduced equipment failures, improved efficiency, and cost savings.
________________________________________
## 3. Everyday Functioning Prompts

Prompt Example:

“Explain how IoT in manufacturing is similar to using smart home devices like smart lights or smart thermostats in daily life.”
Expected Response:
•	Relatable comparison:
o	Industrial IoT sensors = smart home devices
o	Predictive maintenance = notifications/alerts
o	Automation = convenience and efficiency
________________________________________

## 4. Universal Prompt Structures

Prompt Example:
“Explain predictive maintenance in manufacturing using IoT in simple terms with an example.”
Expected Response:
•	Step-by-step explanation:
o	Sensors collect machine data
o	System predicts failures
o	Alerts manager
o	Machine serviced before breakdown
________________________________________

## 5. Prompt Size Limitations

Prompt Example:

“Explain the role of IoT in manufacturing automation with case studies of automotive, electronics, and food processing sectors.”
Refinement into smaller prompts:
•	“Explain IoT-based automation in the automotive industry.”
•	“Explain IoT-based automation in the electronics industry.”
•	“Explain IoT-based automation in the food processing industry.”
Expected Response:
•	Separate detailed explanations for each sector, avoiding information overload.
